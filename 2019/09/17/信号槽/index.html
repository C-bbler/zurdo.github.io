Posted on 2019-09-17

信号槽

信号是以广播形式进行传播，存在需要被处理的信号时，槽函数进行绑定接收。
利用connect关键字进行连接
connect（信号发生的对象，发生的信号（又称发生的动作事件），信号接收者，信号处理（槽函数））
eg：点击一个按钮，然后关闭该窗口
Connect(btu,&QPushButton::click,this,&widget::close)
自定义槽和信号的时候，需要在Signal和Slot后添加相应的函数，但是信号可以只声名而不进行实现，槽函数必须在同名 .cpp文件中实现。
对于槽函数的重载问题
槽函数的参数个数不同，在连接的时候会出现参数无法识别的问题，因为本质上连接的函数的地址，所以使用函数的指针来进行指向函数的位置。
函数指针的写法为： void (作用域， *指针名)（参数类型） = 函数取地址
以上述为例则函数指针为 void (QPushButton:: *pointer) (void) = &QPushButton::click

  1. 一个信号可以连接多个槽函数
  2. 信号可以连接信号，并且信号可以断开和槽函数的连接（disconnect）
  3. 多个信号可以连接同一个槽函数
  4. 信号和槽函数的参数必须是对应的，（信号的参数可以多余，但是槽函数的参数不能多余，同时每一个位置的信号参数和槽参数都必须一致）

Lambda表达式

一般格式
[](){}()；
[]是lambda表达式开始的标识符
其中{}是一个函数，最后再加一个()是函数调用，才能正常使用
[]中存在参数 （空，=，&，a）
其中 空 是指所在的范围没有任何函数对象参数
= 是表示可以使用lambda中的所有可见局部变量包括本身的this，不过该参数是值传递的形式（最常用的）
& 和=类似，不过是引用传递的形式,有一点不同的地方是引用默认内部变量不可修改，如果修改了内容程序就会停住运行
A 是指对某个变量单独引用，在lambda函数体内只能使用者一个参数，也是值传递形式
修改值传递的方式需要用mutable关键字进行修饰
[=]()mutable{m = 10+10;qDebug()<<m;}
存在返回值的时候
[]()->返回值的类型{}
eg：存在int类型的返回值数据
Int rec = []()->int{return 10;}();

QMainWindow

包含一个菜单栏（menu bar），多个工具栏（toolBar）
menuBar（）返回的是一个QMenuBar *bar的对象
将bar放入到窗体中使用this->setMenuBar（bar）
文件栏是addMenu（”文件”），
添加到菜单条是 Qmenu *fileMenu = bar->addMenu("文件")；

工具栏是QToolBar，需要添加在本窗口内，所以指定器父类为this
默认状态下工具栏可以进行拖动，设置不可拖动是setFloatable（bool ）
addToolBar(默认停靠位置，toolbar)
工具栏可以添加菜单项和自定义的工具

状态栏 Qstatus 只能有一个
Qstatus * status= StatusBar（）；
状态栏创建成功后需要添加到当前的显示界面中及即setStatusBar（status）；
但是默认的状态栏没有显示信息即需要手动添加控件来显示相对应的信息，
Qlabel *label = new Qlabel();默认显示在窗口的左侧，但是需要添加到对应的状态栏中
即status->addWidget(label);

铆接部件（浮动窗口）围绕着核心部件QTextEdit
QDockWidget *dock = new QDockWidget（）；
默认是在上层窗口，需要手动更改显示区域 addDockWidget(显示区域，dock);

核心部件只能有一个
QTextEdit *edit = new QTextEdit;
setCentralWidget(edit);
